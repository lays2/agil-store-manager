# Gerenciamento de Produtos – AgilStore

Sistema em Node.js para gerenciamento de inventário de uma loja de eletrônicos.

## Funcionalidades
- Adicionar produtos
- Listar produtos
- Atualizar produtos
- Excluir produtos
- Buscar produtos
- Persistência de dados em arquivo JSON

## Tecnologias
- Node.js
- JavaScript
- readline-sync
- uuid

## Como executar
1. Clone o repositório
2. Instale as dependências:
   npm install
   npm install readline-sync uuid
4. Execute o sistema:
   npm start
